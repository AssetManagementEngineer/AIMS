Creating the data dictionary

When you create a new asset you are creating it in the confines of the data dictionary. So it's important to set the classification structure first.

A note about classification

The Systems Classification Hierarchy (SCH), that determines the system an asset belongs to, works in harmony with the Object Type Library (OTL) that defines the asset ontology. The OTL will have a relationship entity with the SCH in the AIMS database before asset creation.

Asset Registration

It is presumed that the following guidelines are for information only considering the objective of EWR in Digital Engineering is to register the assets in a fully automated procedure. This guide will present the manual process for classification creation along with the technical alternative utilising the ETL provision via CSV and XML as an alternative method that can be triggered manually or automated.
